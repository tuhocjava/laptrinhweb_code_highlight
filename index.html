Code Highlight

Thư viện tham khảo: PrismJS

Các bước thực hiện để có được một đoạn code highlight đẹp mắt.

  1. Thêm đoạn code sau trong cặp thẻ <head></head>: 
          
      <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/prismjs@1.29.0/themes/prism-okaidia.min.css">   
          
    
          <head>
            ...
            <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/prismjs@1.29.0/themes/prism-okaidia.min.css">   
          </head>


  2. Trong đoạn code muốn thực hiện highlight thêm class ="language-html" vào thẻ mở của thẻ code

      <code class="language-html">Đoạn code cần thực hiện highlight</code>
  3. Thêm đoạn code sau vào trước thẻ đóng </body>: 
    
      <script src="https://cdn.jsdelivr.net/npm/prismjs@1.29.0/components/prism-core.min.js"></script>
      <script src="https://cdn.jsdelivr.net/npm/prismjs@1.29.0/plugins/autoloader/prism-autoloader.min.js"></script>

      
    <body>
      ...
      <script src="https://cdn.jsdelivr.net/npm/prismjs@1.29.0/components/prism-core.min.js"></script>
      <script src="https://cdn.jsdelivr.net/npm/prismjs@1.29.0/plugins/autoloader/prism-autoloader.min.js"></script>
    </body>

    4. Vậy là bạn đã có được đoạn code như ý mình rồi nhé!

    Đoạn code trước khi highlight

    Tôi đang học thẻ <em></em> và thẻ <strong></strong>
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Bài 22: Code Highlight_Làm nổi bật code</title>
    
        <link
          href="https://{{cdn}}/prismjs@v1.x/themes/prism.css"
          rel="stylesheet"
        />
    </head>

    Đoạn code sau khi highlight
